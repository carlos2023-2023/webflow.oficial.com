GUÍA TÉCNICA

Cómo conectar
una base de datos

Esta guía explica las opciones más populares para agregar una base de datos a tu proyecto web. Desde soluciones gratuitas hasta opciones empresariales.

1. OPCIONES DISPONIBLES

🔥
Firebase (Firestore)
RECOMENDADO

Base de datos NoSQL en tiempo real de Google. Sin servidor, fácil de integrar con HTML/JS puro. Perfecta para formularios de contacto, reservas y paneles admin.

Gratis hasta: 50k lecturas / 20k escrituras por día.

🐘
Supabase (PostgreSQL)
POPULAR

Alternativa open-source a Firebase con PostgreSQL. Incluye autenticación, storage de archivos y API REST automática. Ideal para proyectos con más estructura.

Gratis hasta: 500MB de base de datos, 2GB de storage.

🍃
MongoDB Atlas
AVANZADO

Base de datos NoSQL en la nube. Perfecta para proyectos con datos complejos y consultas avanzadas. Requiere un backend (Node.js, Express).

Gratis hasta: 512MB de almacenamiento.

2. FIREBASE — INSTALACIÓN PASO A PASO

1

Crear proyecto en Firebase

Ve a console.firebase.google.com → "Agregar proyecto" → sigue el asistente (sin Analytics está bien).

2

Activar Firestore Database

En el panel izquierdo → "Firestore Database" → "Crear base de datos" → Modo de prueba (30 días gratis sin reglas).

3

Obtener las credenciales

Configuración del proyecto (⚙) → "Tus apps" → Agrega app web → Copia el objeto firebaseConfig.

4

Agregar al HTML

Agrega los scripts de Firebase a tu página y copia tu configuración.

<!-- En tu index.html antes del cierre de </body> -->
<script type="module">
  import { initializeApp } from "https://www.gstatic.com/firebasejs/10.7.0/firebase-app.js";
  import { getFirestore, collection, addDoc, getDocs, query, orderBy }
    from "https://www.gstatic.com/firebasejs/10.7.0/firebase-firestore.js";

  // Tu configuración (de la consola de Firebase)
  const firebaseConfig = {
    apiKey: "AIzaSy...",
    authDomain: "tu-proyecto.firebaseapp.com",
    projectId: "tu-proyecto",
    storageBucket: "tu-proyecto.appspot.com",
    messagingSenderId: "123456",
    appId: "1:123456:web:abcdef"
  };

  const app = initializeApp(firebaseConfig);
  const db  = getFirestore(app);

  // ✅ GUARDAR mensaje del formulario de contacto
  async function guardarMensaje() {
    const nombre  = document.getElementById('nombre').value;
    const correo  = document.getElementById('correo').value;
    const mensaje = document.getElementById('mensaje').value;

    await addDoc(collection(db, "mensajes"), {
      nombre, correo, mensaje,
      fecha: new Date().toISOString(),
      leido: false
    });

    alert('Mensaje guardado en la base de datos ✓');
  }

  // ✅ LEER mensajes en el panel admin
  async function cargarMensajes() {
    const q = query(collection(db, "mensajes"), orderBy("fecha", "desc"));
    const snap = await getDocs(q);
    snap.forEach(doc => {
      console.log(doc.id, doc.data());
    });
  }

  // ✅ GUARDAR reserva de hotel
  async function guardarReserva(datos) {
    await addDoc(collection(db, "reservas"), {
      ...datos,
      estado: "pendiente",
      createdAt: new Date()
    });
  }

  // Exponer funciones globalmente
  window.guardarMensaje = guardarMensaje;
  window.cargarMensajes = cargarMensajes;
</script>

3. SUPABASE — CONFIGURACIÓN RÁPIDA

// 1. Crear cuenta en supabase.com y nuevo proyecto
// 2. Ir a Settings → API → copiar URL y anon key
// 3. Crear tabla en el SQL Editor:

CREATE TABLE mensajes (
  id         BIGSERIAL PRIMARY KEY,
  nombre     TEXT NOT NULL,
  correo     TEXT NOT NULL,
  mensaje    TEXT,
  creado_en  TIMESTAMP DEFAULT NOW()
);

// 4. En tu JS:
import { createClient } from 'https://esm.sh/@supabase/supabase-js@2';

const supabase = createClient(
  'https://tu-proyecto.supabase.co',
  'tu-anon-key-publica'
);

// Insertar
const { error } = await supabase
  .from('mensajes')
  .insert({ nombre, correo, mensaje });

// Leer
const { data } = await supabase
  .from('mensajes')
  .select('*')
  .order('creado_en', { ascending: false });

5. SEGURIDAD — REGLAS DE FIRESTORE

// firestore.rules — Reglas básicas de seguridad

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {

    // Solo escritura pública (formularios)
    match /mensajes/{doc} {
      allow write: if true;
      allow read: if request.auth != null;  // Solo admins autenticados
    }

    // Proyectos solo para admins
    match /proyectos/{doc} {
      allow read, write: if request.auth != null;
    }
  }
}
